# YakuakeController

Ruby gem to interact with [Yakuake] through DBus.

## Installation

Add this line to your application's Gemfile:

    gem 'yakuake_controller'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yakuake_controller
